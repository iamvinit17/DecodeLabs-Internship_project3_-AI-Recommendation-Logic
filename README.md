# AI Movie Recommendation System

A simple AI recommendation system developed in Python.

## Features

- Takes user preferences
- Matches movies using recommendation logic
- Displays top recommendations
- Sorts recommendations by similarity score

## Technologies

- Python 3

## How to Run

```bash
python main.py
```

## Sample Input

Genre: Action

Language: English

## Sample Output

1. Avengers: Endgame

2. The Dark Knight

3. Inception
